Cara ambil file project itu : 
1. di clone dulu dengan cara ketik ini di terminal
    git clone link:/repo.git
2. Trus kalian arahin ke file yang udah kalian clone, edit edit lah itu 
3. Trus setelah kalian ngedit ataupun setiap kalian edit jangan lupa git 
    a. git add . 
    b. git commit -M "pesan commit (tolong setiap commit kasih tanggal commit)"
    c. git remote add ke repo yang udah aku buat itu sesuai dengan path nya be or fe 
    d. git push origin main
